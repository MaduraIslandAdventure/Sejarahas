Ujian Sejarah Pro (50 Soal HOTS)

Proyek ini adalah aplikasi kuis interaktif berbasis web (HTML, Tailwind CSS, dan JavaScript murni) yang dirancang untuk menguji kemampuan literasi dan analisis siswa dalam mata pelajaran Sejarah, khususnya pada materi Asas dan Metode Sejarah (Higher Order Thinking Skills - HOTS).

Aplikasi ini memiliki 50 soal dengan variasi tipe (Pilihan Ganda Tunggal, Pilihan Ganda Kompleks, dan Benar/Salah Kompleks).

Fitur Utama

50 Soal HOTS: Soal disusun berdasarkan 13 kisi-kisi utama materi Sejarah.

Variasi Tipe Soal: Menguji pemahaman secara mendalam dengan berbagai format pertanyaan.

Penyimpanan Jawaban: Jawaban yang sudah dikunci (dijawab) akan tersimpan saat navigasi menggunakan tombol Kembali atau Selanjutnya.

Feedback Instan: Jawaban benar atau salah akan ditampilkan langsung beserta pembahasannya setelah tombol Kunci Jawaban ditekan.

Desain Responsif: Menggunakan Tailwind CSS, tampilan kuis akan optimal di perangkat desktop maupun mobile.

Kode Tunggal: Seluruh logika (HTML, CSS, JavaScript) berada dalam satu file (index.html), mempermudah deploy.

Cara Menggunakan (Deploy ke GitHub Pages)

Karena proyek ini hanya terdiri dari satu file HTML/JS/CSS, Anda dapat menjalankannya langsung di browser atau melalui GitHub Pages.

Buat Repositori: Buat repositori baru di akun GitHub Anda (misalnya: kuis-sejarah-hots).

Unggah File: Unggah file index.html ini ke repositori Anda.

Aktifkan GitHub Pages:

Buka Settings pada repositori Anda.

Pilih Pages di menu samping.

Pada bagian Source, pilih branch utama (main atau master) dan folder /(root).

Klik Save.

Akses Kuis: Tunggu beberapa saat, kuis akan tersedia di alamat https://[username Anda].github.io/[nama repositori].

Teknologi

HTML5: Struktur dasar.

JavaScript (ES6): Logika kuis dan manajemen state.

Tailwind CSS (via CDN): Styling dan desain responsif.

Lucide Icons (via CDN): Ikonografi.

Proyek dibuat untuk tujuan edukasi dan pelatihan.
